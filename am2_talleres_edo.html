Taller 10 — Teoría completa de EDO de primer orden

¿Qué es una EDO?

Contiene una función desconocida de una sola variable independiente y sus derivadas. Si hay derivadas respecto a 2 o más variables, es una EDP.
Orden: la derivada de mayor orden que aparece. y'' - 3y = 5x³ → orden 2
Grado: el exponente al que está elevada la derivada de mayor orden. y'' + 3y' = x⁵ → grado 5
Linealidad: los exponentes de y y sus derivadas deben ser 1. Las funciones coeficiente dependen solo de x.

Homogénea vs completa

Homogénea: f(x) = 0 → no hay término independiente
Completa: f(x) ≠ 0 → hay término independiente
También: incompleta / completa

Tipos de solución

General (SG): contiene constante arbitraria C
Particular (SP): C determinada por cond. inicial
Singular (SS): verifica la EDO pero no pertenece a la SG. Al graficar es la envolvente de la SG.

Familia de curvas → EDO

Derivar y eliminar la constante c. La EDO no contiene c.
y(x) = cx² → y' = 2cx → como c = y/x² → EDO: y' = 2y/x
Este proceso siempre tiene tantas constantes irreducibles como sea el orden de la ecuación.

Campo de direcciones e isóclinas

Campo de direcciones: en cada punto se traza un segmento orientado que representa la pendiente de la tangente.
Isóclinas: curvas donde la pendiente es constante k. Para y' = 2y/x → isóclinas: y = kx/2
Trayectorias ortogonales: se invierte y cambia de signo la pendiente. Si la familia tiene y' = f(x,y), las ortogonales tienen y' = -1/f(x,y)
Flia y' = 2y/x → ortogonales: y' = -x/(2y)

Problema de Valor Inicial (PVI)

La curva solución es la gráfica de la solución en el intervalo de definición. Puede diferir del dominio de la función.
El PVI agrega condiciones iniciales a la EDO para obtener la SP.
Teorema de existencia y unicidad: si f(x,y) y ∂f/∂y son continuas en una región R, existe una única solución.

Separación de variables — EDO homogénea (f(x) = 0)

Para: y'(x) + p(x)·y(x) = 0
dy/dx = -p(x)·y → dy/y = -p(x)dx
Integrar ambos miembros:
ln|y| = -∫p(x)dx + C
y(x) = C₀ · e^(-∫p(x)dx)
Al separar variables, los divisores podrían ser cero → puede aparecer la solución singular.

Factor integrante — EDO completa (f(x) ≠ 0)

Para: y' + P(x)·y = f(x)
1. Llevar a forma estándar (dividir por el coeficiente de y')
2. Calcular el factor integrante μ:
μ(x) = e^(∫P(x)dx)
3. Multiplicar ambos lados por μ. El miembro izquierdo se convierte en derivada del producto:
d(μ·y)/dx = f(x)·μ
4. Integrar y despejar y:
y(x) = (1/μ) · (∫f(x)·μ dx + C)
Solución general: y = yc + yp (complementaria + particular)

Ecuaciones exactas

M(x,y)dx + N(x,y)dy = 0 es exacta si ∂M/∂y = ∂N/∂x
1. Verificar la condición de simetría ∂M/∂y = ∂N/∂x
2. Integrar M(x,y) respecto a x → f(x,y) = ∫M dx + φ(y)
3. Derivar respecto a y e igualar a N → hallar φ'(y)
4. Integrar φ'(y). Solución implícita: f(x,y) = C

Ecuación autónoma

dy/dx = f(y) — la variable x no aparece explícitamente
Puntos críticos: valores c donde f(c) = 0. También llamados puntos de equilibrio.
Ejemplo: dP/dt = P(a-bP), a,b > 0. Puntos críticos: P(t)=0 y P(t)=a/b

Desintegración radiactiva

dn/dt = -kn → n(t) = n₀·e^(-kt)

Ejemplo resuelto ↓
Cráneo con 1/6 del C14 original → M(t) = (1/6)M₀
e^(-t·ln2/5730) = 1/6 → t = 5730·ln6/ln2 = 14.811,84 años

Crecimiento poblacional — modelo de Malthus

dP/dt = kP → P(t) = P₀·e^(kt)
k > 0: crecimiento. k < 0: decaimiento.
Modelo logístico: medio ambiente con capacidad de sustento K (capacidad de carga)
P(t) = C·a / (e^(-at) + C·b)
Ejemplo bacteria ↓

Eliminación de medicamentos

x'(t) = -kx → x(t) = x₀·e^(-kt)
Vida media: tiempo en que x se reduce a la mitad. t = 1/k
Ejemplo pentobarbitol ↓
Vida media 5h → e^(-5k) = 1/2 → k = 0,13863
Perro 50kg, dosis mín = 45mg/kg × 50 = 2250mg en t=1
2250 = x₀·e^(-0,13863) → x₀ = 2584,57 mg

Ley de enfriamiento de Newton

dT/dt = -k(T - Tₘ) → T(t) = C₁·e^(kt) + Tₘ
Tₘ = temperatura del medio (constante). La velocidad de variación de T es proporcional a la diferencia T_cuerpo − T_ambiente.
Ejemplo taza ↓
T(0)=75°C, Tₘ=24°C, T(3)=61°C → C₁=51
61 = 24 + 51·e^(3k) → k = ln(37/51)/3 = -0,1069
T(18) = 24 + 51·e^(-0,1069×18) = 31,44°C
¿Cuándo T=56°C? → t = ln(32/51)/(-0,106) = 4,40 min

Ley de corriente: suma algebraica de corrientes en un nodo = 0
Ley de voltaje: suma algebraica de caídas de voltaje = tensión total suministrada

Circuito RL — 1er orden

Kirchhoff: L·di/dt + R·i = E(t) → EDO 1er orden completa
di/dt + (R/L)·i = E(t)/L
μ = e^(Rt/L) → i(t) = V/R + k·e^(-Rt/L)
Ejemplo V=8V, R=10Ω, L=2H ↓
i(0)=5 → 5 = 8/10 + k → k = 4,2
i(t) = 0,8 + 4,2·e^(-5t)
i(2) ≈ 0,8 A

Circuito RC — 1er orden

Kirchhoff: R·dq/dt + q/C = V → EDO 1er orden completa
dq/dt + q/(RC) = V/R
q(t) = VC + k·e^(-t/RC)
Ejemplo R=20Ω, C=3×10⁻³F, V=100V ↓
dq/dt + q/0,06 = 5
q(t) = 0,015 + k·e^(-t/0,06)

Solución explícita vs implícita

Explícita: y se expresa directamente en términos de x. Ej: y = e^(x²)
Implícita: relación G(x,y)=0 que contiene al menos una función que satisface la EDO. Ej: x²+y²=25

Problema de Valor Inicial — procedimiento completo

dy/dx = f(x,y) , y(x₀) = y₀
1) Hallar la familia n-paramétrica de soluciones
2) Usar la condición inicial para determinar las constantes

Modelo logístico — Malthus mejorado

P'(t) = P(a - bP) , a,b > 0
Capacidad de sustento del medio: K = a/b
Puntos críticos: P(t)=0 y P(t)=a/b
A medida que t→∞, P(t)→a/b (población límite)

Trayectorias ortogonales — procedimiento completo

1. Obtener la EDO de la familia de curvas G(x,y,c)=0
2. Aplicar la condición de perpendicularidad: nueva EDO es y' = -1/f(x,y)
3. Resolver esa nueva EDO → familia de trayectorias ortogonales
Flia y = cx² → y' = 2y/x → ortogonales: y' = -x/(2y)
Aparecen en electrostática, flujo de calor, mapas topográficos.
Taller 11 — Dependencia lineal, Wronskiano y reducción de orden

Dependencia e independencia lineal de funciones

Un conjunto {f₁(x), ..., fₙ(x)} es linealmente dependiente (LD) en I si existen constantes c₁,...,cₙ no todas cero tales que:
c₁f₁(x) + c₂f₂(x) + ... + cₙfₙ(x) = 0 para todo x en I
Si el conjunto no es LD, se dice linealmente independiente (LI):
c₁f₁(x) + ... + cₙfₙ(x) = 0 → c₁ = c₂ = ... = cₙ = 0
Caso de 2 funciones: LI si y solo si una NO es múltiplo constante de la otra.

Ejemplos LI

f₁ = senx, f₂ = cosx → LI
f₁ = x², f₂ = x³ → LI en cualquier intervalo
f₁ = eˣ, f₂ = e²ˣ → LI

Ejemplos LD

f₁ = 2x, f₂ = 6x → LD porque f₂ = 3·f₁
f₁ = senx, f₂ = 5senx → LD
f₁ = x², f₂ = 3x², f₃ = x²+1 → LD

Relación con EDO lineal homogénea de orden n

Una EC diferencial lineal homogénea siempre tiene la solución trivial y = 0.
La superposición de 2 o más soluciones también es solución. Si y₁,...,yₖ son soluciones en I:
y = C₁y₁(x) + C₂y₂(x) + ... + Cₖyₖ(x) también es solución en I

Wronskiano — definición

Dado {f₁(x),...,fₙ(x)} con al menos n-1 derivadas, el Wronskiano es el determinante:
W(f₁,...,fₙ) = |f₁ f₂ ... fₙ | |f₁' f₂' ... fₙ'| |... ...| |f₁⁽ⁿ⁻¹⁾ ... fₙ⁽ⁿ⁻¹⁾|
Para 2 funciones:
W(u₁, u₂) = u₁(x)·u₂'(x) − u₁'(x)·u₂(x)

Criterio del Wronskiano

W ≠ 0 en todo punto de I → las funciones son LI en I
W = 0 en todo punto de I → las funciones son LD en I
Si y₁ e y₂ son soluciones de la EC lineal homogénea de orden 2 en I, entonces: o bien W=0 para todo x en I, o bien W≠0 para todo x en I.
No puede ocurrir que W sea cero en algunos puntos y no cero en otros dentro del mismo intervalo.

Cálculo del Wronskiano — ejemplos

Ej 1: y₁=e^(r₁x), y₂=e^(r₂x) ↓
W = e^(r₁x)·r₂e^(r₂x) − r₁e^(r₁x)·e^(r₂x)
W = e^((r₁+r₂)x)·(r₂ − r₁)
Si r₁ ≠ r₂ → W ≠ 0 → son LI
Ej 2: y₁=senx, y₂=cosx ↓
W = senx·(-senx) − cosx·cosx
W = -sen²x − cos²x = -1 ≠ 0
→ son LI
Ej 3: y₁=x², y₂=x³ ↓
W = x²·3x² − 2x·x³ = 3x⁴ − 2x⁴ = x⁴
W ≠ 0 para x ≠ 0 → LI en cualquier intervalo que no contenga 0

Conjunto fundamental de soluciones

Sea una EC lineal homogénea de orden n. Un conjunto fundamental de soluciones es un conjunto de n soluciones LI {y₁,...,yₙ} en I.
Teorema de existencia: siempre existe un conjunto fundamental.
Solución general: si {y₁,...,yₙ} es un conjunto fundamental en I, entonces:
yₘ(x) = C₁y₁(x) + C₂y₂(x) + ... + Cₙyₙ(x)

Condición para que sea conjunto fundamental

Si y₁,...,yₙ son n soluciones de la EC homogénea en I, el conjunto es fundamental si y solo si:
W(y₁, y₂, ..., yₙ) ≠ 0 para todo x en I
En la práctica: calcular el Wronskiano. Si W ≠ 0 → son LI → forman conjunto fundamental.

EC no homogénea — estructura de la solución general

Para la EC completa L(y) = g(x), la solución general es:
y = yc + yp
yc = función complementaria = SG de la EC homogénea asociada L(y) = 0
yp = solución particular = cualquier solución libre de parámetros arbitrarios
Al imponer condiciones iniciales sobre la SG se obtiene la SP (PVI).

Reducción de orden — idea central

Si se conoce una solución no trivial y₁(x) de la EC homogénea de orden 2, se puede hallar una segunda solución y₂ LI con y₁.
Se supone y₂(x) = u(x)·y₁(x) con u(x) incógnita. Como y₂/y₁ ≠ constante → y₂ e y₁ son LI.

Caso 1 — variable dependiente ausente

EC de la forma: F(x, y', y'') = 0, donde falta y explícitamente.
Sustitución: u = y', entonces u' = y''
F(x, u, u') = 0 → EDO de 1er orden en u
Se resuelve para u(x), luego se integra: y = ∫u dx + C
Ejemplo: xy'' + y' = 0 ↓
Sea u = y' → xu' + u = 0
du/u = -dx/x → ln|u| = -ln|x| + C
u = C₁/x → y' = C₁/x
y = C₁·ln|x| + C₂

Caso 2 — hallar y₂ a partir de y₁ conocida

EC estándar: y'' + P(x)y' + Q(x)y = 0, con y₁ conocida
Se propone y₂ = u(x)·y₁(x) → se define w = u' → EC de 1er orden en w
Fórmula directa para y₂:
y₂ = y₁(x) · ∫ [e^(-∫P(x)dx) / y₁²(x)] dx
Con C₁=1 y C₂=0 se obtiene la segunda solución independiente limpia.
Ejemplo: y₁ = eˣ, EC: y'' - y = 0 ↓
P(x) = 0 → e^(-∫0 dx) = 1
y₂ = eˣ · ∫e^(-2x) dx = eˣ · (-½e^(-2x))
y₂ = e^(-x)
SG: y = C₁eˣ + C₂e^(-x)

Términos TET y TEP

TET (estado transitorio): parte que tiende a 0 cuando t → ∞. Generalmente exponenciales decrecientes.
TEP (estado permanente): parte que permanece cuando t → ∞. Generalmente funciones trigonométricas o constantes.
y = yc + yp → yc = TET (desaparece), yp = TEP (permanece)

Operadores diferenciales

El operador D transforma una función diferenciable en su derivada: Dy = dy/dx
D²y = d²y/dx² → Dⁿy = dⁿy/dxⁿ
L tiene propiedad de linealidad: L(αf + βg) = αL(f) + βL(g)
EC lineal en notación compacta: L(y) = 0 (homogénea), L(y) = g(x) (completa)
Ejemplo: y'' + 5y' + 6y = 5x - 3 ↓
En notación de operadores: D²y + 5Dy + 6y = 5x - 3
→ (D² + 5D + 6)y = 5x - 3
→ L(y) = 5x - 3 donde L = D² + 5D + 6
Taller 12 — Movimiento Armónico Simple, amortiguamiento y circuito RLC

Movimiento Armónico Simple — modelo físico

Sistema: resorte + masa "m". La fuerza impulsora actúa sobre la masa.
Ley de Hooke: el resorte ejerce una fuerza de restitución proporcional al alargamiento:
F = −k·s
2da Ley de Newton:
W = mg = ks → my''(t) + ky(t) = 0

Ecuación auxiliar y solución general

Raíces complejas conjugadas (m, k positivos):
r² + ω² = 0 → r₁,₂ = ±ωi donde ω = √(k/m)
y(t) = C₁cos(ωt) + C₂sen(ωt)
Condición inicial: C₁ = y(0), C₂ = y'(0)/ω

T = 2π/ω [período]

Resonancia pura (γ → ω, sin amortiguamiento)

Cuando γ → ω, la amplitud crece sin límite. Aplicando L'Hôpital:
x(t) = F₀/(2ω²) · (−sen(ωt) + ωt·cos(ωt))
Cuando t → ∞, los desplazamientos crecen sin límite.
Sintonización de frecuencias: la frecuencia impulsora γ coincide con la natural ω → resonancia.

Resonancia con amortiguamiento

La amplitud de Yp depende de γ:
Amplitud Yp = F₀ / √((ω²-γ²)² + (2λγ)²)
Se maximiza derivando respecto a γ e igualando a 0:
γ_resonante = √(ω² − 2λ²)
Solo existe γ_resonante real si ω² > 2λ² (sistema subamortiguado suficientemente débil).

Pulsación y modulación

Pulsación: fenómeno cuando 2 frecuencias son próximas entre sí. La amplitud oscila lentamente.
Modulación de amplitud: mismo fenómeno en electrónica, utilizado para afinar instrumentos.
Sistema subamortiguado → grandes oscilaciones cuando frecuencia del rozamiento ≥ resonancia del sistema.
