Teoría completa Taller 11 — Dependencia lineal, Wronskiano y reducción de orden

Dependencia e independencia lineal de funciones

Un conjunto de funciones {f₁(x), f₂(x), ..., fₙ(x)} es linealmente dependiente en un intervalo I si existen constantes c₁, c₂, ..., cₙ no todas cero tales que:
c₁f₁(x) + c₂f₂(x) + ... + cₙfₙ(x) = 0 para todo x en I
Si el conjunto no es linealmente dependiente, se dice linealmente independiente (LI):
c₁f₁(x) + c₂f₂(x) + ... + cₙfₙ(x) = 0 → c₁ = c₂ = ... = cₙ = 0
Caso de 2 funciones: LI si y solo si una NO es múltiplo constante de la otra.

Ejemplos LI

f₁ = senx, f₂ = cosx → LI (ninguna es múltiplo de la otra)
f₁ = x², f₂ = x³ → LI en cualquier intervalo
f₁ = eˣ, f₂ = e²ˣ → LI

Ejemplos LD

f₁ = 2x, f₂ = 6x → LD porque f₂ = 3·f₁
f₁ = senx, f₂ = 5senx → LD
f₁ = x², f₂ = 3x², f₃ = x²+1 → LD

Relación con EDO lineal homogénea de orden n

Una EC diferencial lineal homogénea siempre tiene la solución trivial y = 0.
La superposición de 2 o más soluciones de una EC lineal homogénea también es solución.
Si y₁, y₂, ..., yₖ son soluciones en I, entonces la combinación lineal:
y = C₁y₁(x) + C₂y₂(x) + ... + Cₖyₖ(x)
también es solución en I. Un múltiplo constante de una solución también es solución.

Wronskiano — definición

Dado un conjunto de funciones {f₁(x), f₂(x), ..., fₙ(x)} con al menos n-1 derivadas, el Wronskiano es el determinante:
W(f₁, f₂, ..., fₙ) = |f₁ f₂ ... fₙ | |f₁' f₂' ... fₙ'| |... ...| |f₁⁽ⁿ⁻¹⁾ ... fₙ⁽ⁿ⁻¹⁾|
Para 2 funciones, el Wronskiano es simplemente:
W(u₁, u₂) = u₁(x)·u₂'(x) − u₁'(x)·u₂(x)

Criterio del Wronskiano

W ≠ 0 en todo punto de I → las funciones son LI en I
W = 0 en todo punto de I → las funciones son LD en I (o no se puede concluir si W=0 en algún punto)
Si y₁ e y₂ son soluciones de la EC lineal homogénea de orden 2 en I, entonces:
O bien W(y₁,y₂) = 0 para todo x en I, o bien W(y₁,y₂) ≠ 0 para todo x en I.
No puede ocurrir que W sea cero en algunos puntos y no cero en otros dentro del mismo intervalo.

Cálculo del Wronskiano — ejemplos

Ej 1: y₁=e^(r₁x), y₂=e^(r₂x) ↓
W = e^(r₁x)·r₂e^(r₂x) − r₁e^(r₁x)·e^(r₂x)
W = e^((r₁+r₂)x)·(r₂ − r₁)
Si r₁ ≠ r₂ → W ≠ 0 → son LI
Ej 2: y₁=senx, y₂=cosx ↓
W = senx·(-senx) − cosx·cosx
W = -sen²x − cos²x = -1 ≠ 0
→ son LI
Ej 3: y₁=x², y₂=x³ ↓
W = x²·3x² − 2x·x³ = 3x⁴ − 2x⁴ = x⁴
W ≠ 0 para x ≠ 0 → LI en cualquier intervalo que no contenga 0

Conjunto fundamental de soluciones

Sea una EC lineal homogénea de orden n. Un conjunto fundamental de soluciones es un conjunto de n soluciones LI {y₁, y₂, ..., yₙ} en I.
Teorema de existencia: siempre existe un conjunto fundamental de soluciones de la EC lineal homogénea de orden n en I.
Solución general: si {y₁, y₂, ..., yₙ} es un conjunto fundamental en I, entonces:
yₘ(x) = C₁y₁(x) + C₂y₂(x) + ... + Cₙyₙ(x)
con C₁, C₂, ..., Cₙ constantes arbitrarias, es la solución general en I.

Condición para que sea conjunto fundamental

Sean aₙ(x), aₙ₋₁(x), ..., a₀(x) y g(x) continuas en I, con aₙ(x) ≠ 0 en I.
Si y₁, y₂, ..., yₙ son n soluciones de la EC homogénea en I, entonces el conjunto es fundamental si y solo si:
W(y₁, y₂, ..., yₙ) ≠ 0 para todo x en I
En la práctica: calcular el Wronskiano. Si W ≠ 0 → son LI → forman conjunto fundamental.

EC. no homogénea — estructura de la solución general

Para la EC completa L(y) = g(x), la solución general es:
y = yc + yp
yc = función complementaria = SG de la EC homogénea asociada L(y) = 0
yp = solución particular = cualquier solución libre de parámetros arbitrarios que satisfaga L(y) = g(x)
Al imponer condiciones iniciales sobre la SG y su derivada se calculan los valores específicos de C₁ y C₂ → se obtiene la SP (PVI).

Reducción de orden — idea central

Si se conoce una solución no trivial y₁(x) de la EC homogénea de orden 2, se puede hallar una segunda solución y₂ LI con y₁.
Se supone y₂(x) = u(x)·y₁(x) con u(x) incógnita. Como y₂/y₁ ≠ constante → y₂ e y₁ son LI.
Se sustituye en la EC para obtener una EC de 1er orden en w = u':
g(x)·w'' + h(x)·w' = 0 → EDO 1er orden: variable dependiente ausente

Caso 1 — variable dependiente ausente

EC de la forma: F(x, y', y'') = 0, donde falta y explícitamente.
Sustitución: u = y', entonces u' = y''
F(x, u, u') = 0 → EDO de 1er orden en u
Se resuelve para u(x), luego se integra para obtener y = ∫u dx + C
Ejemplo: xy'' + y' = 0 ↓
Sea u = y' → xu' + u = 0
du/u = -dx/x → ln|u| = -ln|x| + C
u = C₁/x → y' = C₁/x
y = C₁·ln|x| + C₂

Caso 2 — hallar y₂ a partir de y₁ conocida

EC estándar: y'' + P(x)y' + Q(x)y = 0, con y₁ conocida y₁ ≠ 0
Se propone y₂ = u(x)·y₁(x). Se calcula y₂' y y₂'' y se sustituye:
Se define w = u' → se obtiene EC de 1er orden separable en w:
dw/w + (2y₁'/y₁ + P)dx = 0
Integrando:
w·y₁² = C·e^(-∫P dx)
La fórmula directa para y₂ es:
y₂ = y₁(x) · ∫ [e^(-∫P(x)dx) / y₁²(x)] dx
Con C₁=1 y C₂=0 se obtiene la segunda solución independiente limpia.
Ejemplo: y₁ = eˣ, EC: y'' - y = 0 ↓
P(x) = 0 → e^(-∫0 dx) = 1
y₂ = eˣ · ∫ [1/e^(2x)] dx = eˣ · ∫e^(-2x) dx
y₂ = eˣ · (-1/2)e^(-2x) = -½e^(-x)
→ y₂ = e^(-x) (eliminando constante)
SG: y = C₁eˣ + C₂e^(-x)

Términos TET y TEP

Término de estado transitorio (TET): parte de la solución que tiende a 0 cuando t → ∞. Generalmente son las funciones exponenciales decrecientes.
Término de estado permanente (TEP): parte que permanece constante o invariable cuando t → ∞. Generalmente son funciones trigonométricas o constantes.
y = yc + yp → yc = TET (desaparece), yp = TEP (permanece)

Operadores diferenciales

El operador D transforma una función diferenciable en su derivada: Dy = dy/dx
D²y = d²y/dx² → Dⁿy = dⁿy/dxⁿ
El operador diferencial de orden n: L = aₙ(x)Dⁿ + aₙ₋₁(x)Dⁿ⁻¹ + ... + a₁(x)D + a₀(x)
L tiene propiedad de linealidad: L(αf + βg) = αL(f) + βL(g)
EC lineal en notación compacta: L(y) = 0 (homogénea), L(y) = g(x) (completa)
Ejemplo: y'' + 5y' + 6y = 5x - 3 ↓
En notación de operadores: D²y + 5Dy + 6y = 5x - 3
→ (D² + 5D + 6)y = 5x - 3
→ L(y) = 5x - 3 donde L = D² + 5D + 6
