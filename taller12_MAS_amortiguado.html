Teoría completa Taller 12 — Movimiento Armónico Simple, sistema masa-resorte amortiguado y libre

Movimiento Armónico Simple — modelo físico

Sistema: resorte + masa "m" suspendidos. La fuerza impulsora actúa sobre la masa.
Ley de Hooke: el resorte ejerce una fuerza de restitución proporcional al alargamiento:
F = −k·s
2da Ley de Newton: equilibrio estático y dinámica dan:
W = mg = ks → my''(t) + ky(t) = 0
Desplazamientos y velocidades positivos hacia abajo y derecha. El mov. comienza si al menos una condición inicial ≠ 0.

Ecuación auxiliar y solución general

Se propone y(t) = eʳᵗ. Como m, k son positivos, las raíces son complejas conjugadas:
r² + ω² = 0 → r₁,₂ = ±ωi donde ω = √(k/m)
y(t) = C₁Cos(ωt) + C₂Sen(ωt)
Condición inicial: C₁ = y(0), C₂ = y'(0)/ω

Magnitudes del MAS

Frecuencia angular:
ω = √(k/m)
Período:
T = 2π/ω
Frecuencia:
f = 1/T = ω/2π

Forma alternativa

Expresión senoidal con amplitud y fase:
y(t) = A·Sen(ωt + φ)
A = √(C₁² + C₂²)
tan(φ) = C₁/C₂
φ = ángulo de fase. Elegir cuadrante según signos de C₁ y C₂.

Ángulo de fase — cuadrante correcto

Instantes especiales — equilibrio y máxima extensión

Depende de si la solución está expresada con Cos o con Sen:
Si y(t) = A·Cos(ωt + φ)
Equilibrio (y = 0):
Cos(ωt+φ) = 0
ωt+φ = π/2 + nπ
t = (π/2 − φ + nπ)/ω
Máx. extensión abajo (y = A):
Cos(ωt+φ) = 1
ωt+φ = 2nπ
t = (2nπ − φ)/ω
Máx. extensión arriba (y = −A):
Cos(ωt+φ) = −1
ωt+φ = π + 2nπ
t = (π − φ + 2nπ)/ω
Si y(t) = A·Sen(ωt + φ)
Equilibrio (y = 0):
Sen(ωt+φ) = 0
ωt+φ = nπ
t = (nπ − φ)/ω
Máx. extensión abajo (y = A):
Sen(ωt+φ) = 1
ωt+φ = π/2 + 2nπ
t = (π/2 − φ + 2nπ)/ω
Máx. extensión arriba (y = −A):
Sen(ωt+φ) = −1
ωt+φ = −π/2 + 2nπ
t = (−π/2 − φ + 2nπ)/ω
n = 0, 1, 2, … Tomar el menor t > 0. El primer instante positivo que cumpla la igualdad.

Velocidad máxima y aceleración máxima

Derivar y(t) para obtener velocidad y aceleración:
Con y(t) = A·Cos(ωt + φ)
y'(t) = −Aω·Sen(ωt+φ)
y''(t) = −Aω²·Cos(ωt+φ)
Con y(t) = A·Sen(ωt + φ)
y'(t) = Aω·Cos(ωt+φ)
y''(t) = −Aω²·Sen(ωt+φ)
El módulo máximo de Cos y Sen es 1, por lo tanto:
|v_máx| = A·ω    (cuando el sistema pasa por el equilibrio, y = 0)
|a_máx| = A·ω²   (cuando el sistema está en la extensión máxima, |y| = A)
Relación clave: a_máx = ω² · A = ω · v_máx
Estas fórmulas son válidas tanto para la expresión en Cos como en Sen, solo cambia en qué instante ocurren dentro del ciclo.

Resortes en paralelo y en serie

Paralelo:
k_eq = k₁ + k₂
Serie:
1/k_eq = 1/k₁ + 1/k₂
Ejemplo: m=2 kg, k=50 N/m ↓
ω = √(50/2) = √25 = 5 rad/s
x₀ = −1 m, v₀ = 2 m/s
C₁ = −1, C₂ = 2/5 = 0.4
x(t) = −cos(5t) + 0.4·sen(5t)
Amplitud A = √(1 + 0.16) ≈ 1.08 m
Período T = 2π/5 ≈ 1.26 s
Frecuencia f = 5/2π ≈ 0.8 Hz

Energía del oscilador armónico simple

Sistema conservativo → energía total constante:
Et = Ec + Ep
Ec = ½m·(y'(t))² → Ep = ½k·(y(t))²
Ep_máxima = ½kA² cuando Ec = 0

Movimiento amortiguado y libre

En sistemas reales actúan fuerzas disipativas (fricción) que retardan el mov. La energía mecánica disminuye con el tiempo.
Modelo con amortiguamiento (β > 0):
my''(t) + by'(t) + ky(t) = 0
EC auxiliar: mr² + br + k = 0
r₁,₂ = (−b ± √(b²−4km)) / 2m

Forma alternativa — subamortiguado

y(t) = A·e^(αt)·Sen(√(ω²−λ²)·t + φ)
Amplitud amortiguada: A·e^(αt) → 0 cuando t → ∞
Cuasiperiodo: 2π/√(ω²−λ²) — intervalo entre 2 máximos sucesivos
Cuasifrecuencia: √(ω²−λ²) / 2π
La curva admite una asíntota horizontal y = 0 (P.E.).

Movimiento amortiguado y forzado

Se agrega una fuerza externa de excitación f(t) al sistema:
my''(t) + by'(t) + ky(t) = f(t)
Solución general:
Yg(t) = Yh(t) + Yp(t)
Yh = función complementaria (sol. de la ec. homogénea)
Yp = solución particular de la ec. completa

Términos transitorio y permanente

TET (término de estado transitorio): parte de la solución que tiende a 0 cuando t → ∞. Son las funciones exponenciales decrecientes (Yh si λ > 0).
TEP (término de estado permanente): parte que permanece constante o invariable cuando t → ∞. Son las funciones trigonométricas o constantes (generalmente Yp).
y = Yh + Yp → Yh = TET, Yp = TEP

Mov. forzado NO amortiguado

Sin fuerza de amortiguamiento (b=0) y con fuerza periódica f(t) = F₀Sen(γt):
x'' + ω²x = F(t) donde F(t) = f(t)/m
Si γ ≠ ω (frecuencias distintas):
Yp(t) = F₀/(ω²−γ²) · Sen(γt)
X(t) = C₁Cos(ωt) + C₂Sen(ωt) + F₀/(ω²−γ²)·Sen(γt)

Circuito RLC en serie — modelo

Por la 2da Ley de Kirchhoff, la suma de las caídas de voltaje es igual a la tensión total aplicada:
L·di/dt + Ri + (1/C)·q = E(t)
Como i = dq/dt → i' = q'':
L·q'' + R·q' + (1/C)·q = E(t)
EDO lineal de 2° orden completa → Solución: q(t) = qh(t) + qp(t)

Vibraciones eléctricas libres (E(t)=0)

EC auxiliar: Lm² + Rm + 1/C = 0
Factor de decaimiento: e^(−Rt/2L)
Si E(t) = 0 y R = 0: circuito no amortiguado, las vibraciones eléctricas no tienden a 0 → respuesta armónica simple.
Si E(t) ≠ 0 (periódico o constante) y R ≠ 0:
R ≠ 0 → Yh es solución transitoria
Yp es solución de estado estable

Reactancia: X = Lγ − 1/(Cγ)

Resonancia pura (γ → ω, sin amortiguamiento)

Cuando la frecuencia de la fuerza externa γ → frecuencia natural ω, la amplitud crece sin límite.
Se aplica L'Hôpital al límite:
x(t) = lim(γ→ω) F₀(-γSen(ωt)+ωSen(γt)) / ω(ω²-γ²)
= F₀/(2ω²) · (-Sen(ωt) + ωt·Cos(ωt))
Cuando t → ∞, los desplazamientos crecen sin límite.
Sintonización de frecuencias: la frecuencia impulsora γ coincide con la natural ω → pulsación/resonancia.

Resonancia en sistema con amortiguamiento

La frecuencia angular resonante es el valor de γ para el cual la amplitud de la oscilación es máxima.
Para el movimiento forzado y amortiguado la amplitud de Yp depende de γ:
Amplitud Yp = F₀ / √((ω²-γ²)² + (2λγ)²)
Se maximiza derivando respecto a γ e igualando a 0:
γ_resonante = √(ω² - 2λ²)
Solo existe γ_resonante real si ω² > 2λ² (sistema subamortiguado suficientemente débil).

Pulsación y modulación

Pulsación: fenómeno que ocurre cuando 2 frecuencias son próximas entre sí. La amplitud oscila lentamente.
Modulación de amplitud: mismo fenómeno en electrónica, utilizado para afinar instrumentos.
Oscilaciones forzadas → frecuencia de la fuerza exterior, no la del cuerpo.
La respuesta depende de la relación entre frecuencia natural ω y forzada γ.
Si sistema es subamortiguado (Δ < 0) → grandes oscilaciones cuando la frecuencia de la función rozamiento ≥ resonancia del sist.
