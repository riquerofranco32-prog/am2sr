Teoría completa Taller 10 — EDO de primer orden, extraída y cruzada del libro y resumen

¿Qué es una EDO?

Contiene una función desconocida de una sola variable independiente y sus derivadas. Si hay derivadas respecto a 2 o más variables, es una EDP.
Orden: la derivada de mayor orden que aparece. y'' - 3y = 5x³ → orden 2
Grado: el exponente al que está elevada la derivada de mayor orden. y'' + 3y' = x⁵ → grado 5
Linealidad: los exponentes de y y sus derivadas deben ser 1. Las funciones coeficiente dependen solo de x.

Homogénea vs completa

Homogénea: f(x) = 0 → no hay término independiente
Completa: f(x) ≠ 0 → hay término independiente
También: incompleta / completa

Tipos de solución

General (SG): contiene constante arbitraria C
Particular (SP): C determinada por cond. inicial
Singular (SS): verifica la EDO pero no pertenece a la SG. Al graficar es la envolvente de la SG.

Familia de curvas → EDO

Derivar y eliminar la constante c. La EDO no contiene c.
y(x) = cx² → y' = 2cx → como c = y/x² → EDO: y' = 2y/x
Este proceso siempre tiene tantas constantes irreducibles como sea el orden de la ecuación.

Campo de direcciones e isóclinas

Campo de direcciones: en cada punto se traza un segmento orientado que representa la pendiente de la tangente. Provee info sobre el comportamiento de la familia de soluciones.
Isóclinas: curvas donde la pendiente es constante k. Para y' = 2y/x → isóclinas: y = kx/2
Trayectorias ortogonales: se invierte y cambia de signo la pendiente. Si la familia tiene y' = f(x,y), las ortogonales tienen y' = -1/f(x,y)
Flia y' = 2y/x → ortogonales: y' = -x/(2y)

Curva solución y condiciones iniciales

La curva solución es la gráfica de la solución en el intervalo de definición. Puede diferir del dominio de la función.
El Problema de Valor Inicial (PVI) agrega condiciones iniciales a la EDO para obtener la SP.
Teorema de existencia y unicidad: si f(x,y) y ∂f/∂y son continuas en una región R, existe una única solución.

Separación de variables — EDO homogénea (f(x) = 0)

Para: y'(x) + p(x)·y(x) = 0
dy/dx = -p(x)·y → dy/y = -p(x)dx
Integrar ambos miembros:
ln|y| = -∫p(x)dx + C
y(x) = C₀ · e^(-∫p(x)dx)
Atención: al separar variables, los divisores podrían ser cero en algún punto → puede aparecer la solución singular.

Factor integrante — EDO completa (f(x) ≠ 0)

Para: y' + P(x)·y = f(x)
1. Llevar a forma estándar (dividir por el coeficiente de y')
2. Calcular el factor integrante μ:
μ(x) = e^(∫P(x)dx)
3. Multiplicar ambos lados por μ. El miembro izquierdo se convierte en la derivada del producto:
d(μ·y)/dx = f(x)·μ
4. Integrar y despejar y:
y(x) = (1/μ) · (∫f(x)·μ dx + C)
La solución general es: y = yc + yp (solución complementaria + solución particular)

EC. exactas — complemento del libro

M(x,y)dx + N(x,y)dy = 0 es exacta si ∂M/∂y = ∂N/∂x
1. Verificar la condición de simetría ∂M/∂y = ∂N/∂x
2. Integrar M(x,y) respecto a x → f(x,y) = ∫M dx + φ(y)
3. Derivar respecto a y e igualar a N → hallar φ'(y)
4. Integrar φ'(y). Solución implícita: f(x,y) = C
Este método da directamente la familia de curvas de nivel (solución general).

EC. autónoma

dy/dx = f(y) — la variable x no aparece explícitamente
Puntos críticos: valores c donde f(c) = 0. También llamados puntos de equilibrio.
Si y(x) = c es solución constante de la EC autónoma → solución de equilibrio
Ejemplo del libro: dP/dt = P(a-bP), a,b > 0. Puntos críticos: P(t)=0 y P(t)=a/b

Desintegración radiactiva

dn/dt = -kn → n(t) = n₀·e^(-kt)
k > 0 es la constante de decaimiento. k = 0,0001216 para C14, T½ = 5730 años
Modelo vida media: M(t) = M₀·e^(-t·ln2/T)
Para hallar edad: t = ln(c_reliquia/c_actual) / (-k)
Ejemplo resuelto ↓
Cráneo con 1/6 del C14 original → M(t) = (1/6)M₀
e^(-t·ln2/5730) = 1/6 → t = 5730·ln6/ln2 = 14.811,84 años

Crecimiento poblacional — modelo de Malthus

dP/dt = kP → P(t) = P₀·e^(kt)
Cuando k > 0: crecimiento. Cuando k < 0: decaimiento.
Malthus mejorado: tasa de mortalidad proporcional al tamaño. Lím P(t) = a/b (población límite)
P'(t) = P(t)(k - k₁) → solución: P(t) = P₀·e^(kt)
Modelo logístico: medio ambiente con capacidad de sustento K (capacidad de carga)
P(t) = C·a / (e^(-at) + C·b)
Ejemplo bacteria ↓
P₀=1000, se duplica en 1h → P(1)=2000
1000·e^k = 2000 → k = ln2
P(1.5) = 1000·e^(1.0397) = 2828 bacterias
P(t)=4000 → t = ln4/ln2 = 2 horas

Eliminación de medicamentos

x'(t) = -kx → x(t) = x₀·e^(-kt)
Vida media: tiempo en que x se reduce a la mitad. t = 1/k
Concentración en el tiempo: c(t) = a·t·e^(-bt), con a,b > 0
a aumenta la amplitud del pico; b controla si la droga aumenta o disminuye
Ejemplo pentobarbitol ↓
Vida media 5h → e^(-5k) = 1/2 → k = 0,13863
Perro 50kg, dosis mín = 45mg/kg × 50 = 2250mg en t=1
2250 = x₀·e^(-0,13863) → x₀ = 2584,57 mg

Ley de enfriamiento de Newton

dT/dt = -k(T - Tₘ) → T(t) = C₁·e^(kt) + Tₘ
Tₘ = temperatura del medio (constante). k > 0 → calor se disipa en dirección al ambiente.
La velocidad de variación de T es proporcional a la diferencia entre T del cuerpo y T del ambiente.
Ejemplo taza ↓
T(0)=75°C, Tₘ=24°C, T(3)=61°C → C₁=51
61 = 24 + 51·e^(3k) → k = ln(37/51)/3 = -0,1069
T(18) = 24 + 51·e^(-0,1069×18) = 31,44°C
¿Cuándo T=56°C? → t = ln(32/51)/(-0,106) = 4,40 min

Leyes de Kirchhoff

Ley de corriente: suma algebraica de corrientes en un nodo = 0
Ley de voltaje: suma algebraica de caídas de voltaje = tensión total suministrada
Elementos y sus caídas
Resistencia R → Vᴿ = R·i
Inductancia L → Vʟ = L·di/dt
Capacitor C → Vᴄ = (1/C)·q
Unidades
E(t) → voltios (V)
R → ohms (Ω)
L → henrios (H)
C → faradios (F)
q → coulombs; i → amperios

Circuito RL — 1er orden

Kirchhoff: L·di/dt + R·i = E(t) → EDO 1er orden completa
di/dt + (R/L)·i = E(t)/L
Factor integrante: μ = e^(Rt/L)
i(t) = V/R + k·e^(-Rt/L)
Condición inicial i(0) = i₀ → determina k
Ejemplo V=8V, R=10Ω, L=2H ↓
i(0)=5 → 5 = 8/10 + k·e⁰ → k = 4,2
i(t) = 0,8 + 4,2·e^(-10t/2)
i(2) = 0,8 + 4,2·e^(-10) ≈ 0,8 A

Circuito RC — 1er orden

Kirchhoff: R·dq/dt + q/C = V → EDO 1er orden completa
dq/dt + q/(RC) = V/R
Factor integrante: μ = e^(t/RC)
q(t) = VC + k·e^(-t/RC)
Ejemplo R=20Ω, C=3×10⁻³F, V=100V ↓
dq/dt + q/0,06 = 5
Solución: q(t) = 5·3×10⁻³ + k·e^(-t/0,06)
q(t) = 0,015 + k·e^(-t/0,06)

Solución explícita vs implícita

Explícita: la variable dependiente se expresa directamente en términos de x. Ej: y = e^(x²)
Implícita: una relación G(x,y)=0 contiene al menos una función que satisface la EDO en I. Ej: x²+y²=25
La curva solución puede diferir del dominio de la función y del intervalo de definición de la solución.

Problema de valor inicial (PVI)

dy/dx = f(x,y) , y(x₀) = y₀
Se busca una solución definida en algún intervalo I que contenga x₀ y satisfaga la condición inicial.
Para resolverlo: 1) hallar la familia n-paramétrica de soluciones, 2) usar la condición inicial para determinar las constantes.

Ecuación diferencial autónoma y puntos de equilibrio

dy/dx = f(y) — x no aparece explícitamente
Si y = c constante satisface f(c) = 0 → solución de equilibrio (también: punto crítico o estacionario)
En el campo de direcciones, las soluciones de equilibrio son rectas horizontales.
Interpretación: creciente si dy/dx > 0; decreciente si dy/dx < 0

Modelo logístico — Malthus mejorado

P'(t) = P(a - bP) , a,b > 0
Capacidad de sustento del medio: K = a/b
Puntos críticos: P(t)=0 y P(t)=a/b
P(t) = C·a / (e^(-at) + C·b)
A medida que t→∞, P(t)→a/b (población límite)
Agrega término de competencia intraespecífica proporcional al nº de interacciones bipártitas P(P-1)/2

Trayectorias ortogonales — procedimiento completo

1. Obtener la EDO de la familia de curvas G(x,y,c)=0
2. Aplicar la condición de perpendicularidad: la nueva EDO es y' = -1/f(x,y)
3. Resolver esa nueva EDO → solución = familia de trayectorias ortogonales
Flia y = cx² → y' = 2y/x → ortogonales: y' = -x/(2y)
Las familias de trayectorias ortogonales aparecen en electrostática, flujo de calor, mapas topográficos.
